# 🧩 CubeX

CubeX is an interactive learning platform designed to help beginners and enthusiasts learn how to solve a Rubik's Cube through step-by-step tutorials, animations, and visual guidance.

The website breaks down the solving process into easy-to-follow stages, making it simple for anyone to master the cube.

---

## 🚀 Features

- 📚 Step-by-step Rubik's Cube solving guide
- 🎥 Interactive animations for every algorithm
- 🧩 Beginner-friendly learning path
- 📱 Fully Responsive Design
- ⚡ Fast performance with Next.js
- 🎨 Modern UI built with Tailwind CSS

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/CubeX.git
```

Navigate to the project

```bash
cd CubeX
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:3000
```

---

## 📦 Build

```bash
npm run build
```

Run production build

```bash
npm start
```
